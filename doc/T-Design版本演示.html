T-Design版本特色功能

🎨 现代化UI设计

采用T-Design组件库，提供一致性的视觉体验。渐变色背景、圆角设计、阴影效果营造现代感。

📱 响应式布局

按钮自动填满屏幕，支持不同屏幕尺寸。Grid布局确保在各种设备上都有良好的显示效果。

🔊 智能语音播报

集成Web Speech API，支持中文语音播报。语音按钮状态反馈，播报过程中显示loading状态。

📊 进度指示系统

底部步骤指示器显示当前进度，用户可以清楚了解操作流程。支持前进、后退操作。

💾 历史记录功能

自动保存最近的对话记录，支持快速重播。折叠面板设计节省屏幕空间。

♿ 无障碍优化

大字体、高对比度设计，适合视力不佳的用户。按钮足够大，方便操作。

🛠️ 技术规格

Vue 3 + T-Design
响应式Grid布局
Web Speech API
CSS3动画效果
渐变色设计
移动端优化
